# **App Name**: Joblytics

## Core Features:

- Job Search and Display: Display job postings as cards, with details like job title, company name, location, salary, skills, profile match score and posted date.
- Comprehensive Filtering: Implement a filter section at the top of the job search page to allow users to narrow down job results based on various criteria. Use icons from favicon.
- Bulk Apply Selection: Enable users to select multiple job postings via a select box for bulk applying.
- User Profile: Include a user profile picture in the top right corner with a pop-up menu for 'logout' and 'settings' options.
- AI-Powered Job Search: Enable search by AI by incorporating a tool that helps you decide when or if a particular combination of words might result in the user being better-served, enabling the AI to adjust the search in order to produce such improved results, even when some of the entered keywords have weak matches.
- Theme Switch: Implement a light/dark theme switch for user interface preference.

## Style Guidelines:

- Primary color: A calm, professional blue (#507EAD) to convey trust and stability in job searching. 
- Background color: A desaturated light blue (#E8F0F7) to provide a clean and unobtrusive backdrop. 
- Accent color: A soft lavender (#8D60A6) for interactive elements and highlights, complementing the primary color. 
- Body and headline font: 'Inter' for a modern and readable design.
- Use clear and concise icons for job details (location, salary, etc.).
- Job cards with consistent spacing and clear information hierarchy.
- Smooth transitions and subtle hover effects on job cards.
